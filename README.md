# ✨ AR Picture Book — Setup Guide

## Your folder should contain:
- `index.html` ← the web app (already done!)
- `animation.mp4` ← your animation
- `ambient.mp3` ← your forest sounds
- `prototype1.jpg` ← your trigger image
- `targets.mind` ← YOU NEED TO GENERATE THIS (Step 1 below)

---

## STEP 1 — Generate your targets.mind file (2 minutes)

1. Go to: **https://hiukim.github.io/mind-ar-js-doc/tools/compile**
2. Click **"Upload Images"**
3. Upload your `prototype1.jpg`
4. Wait for it to process (green progress bar)
5. Click **"Export"** — this downloads `targets.mind`
6. Put `targets.mind` in this folder ✅

---

## STEP 2 — Host on GitHub Pages (free!)

1. Go to **github.com** → Sign up / Log in
2. Click the **+** button → **New repository**
3. Name it: `ar-picture-book`
4. Make sure it's set to **Public**
5. Click **Create repository**
6. Drag ALL files from this folder into the GitHub page
7. Commit (save) them
8. Go to **Settings → Pages**
9. Under "Source", select **main branch** → Save
10. Your link will be: `https://YOUR-USERNAME.github.io/ar-picture-book`

---

## STEP 3 — Make your QR code (1 minute)

1. Go to **qr.io** or **qrcode-monkey.com**
2. Paste your GitHub Pages link
3. Download the QR code
4. Print it on / next to your book page!

---

## How it works for the reader:
1. They scan QR code → browser opens
2. Tap "Begin Experience"
3. Allow camera
4. Point at the illustration page
5. 🌿 Magic happens!
